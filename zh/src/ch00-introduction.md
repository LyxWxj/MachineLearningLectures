# 计算神经科学与机器学习

## 课程概述

本课程基于 **Neuromatch Academy** 的计算神经科学课程，系统介绍从数学基础到机器学习核心算法，再到计算神经科学前沿主题的完整知识体系。

## 学习路线

本课程分为三个部分：

### 第一部分：基础讲座

| 顺序 | 主题                 | 内容                                             |
| ---- | -------------------- | ------------------------------------------------ |
| 0    | **数学基础**         | 微积分、线性代数、概率论——机器学习的三大数学支柱 |
| 1    | **机器学习基础**     | 什么是机器学习、学习范式、工作流程与核心术语     |
| 2    | **度量、损失与优化** | 评估指标、损失函数、梯度下降与优化器             |
| 3    | **贝叶斯家族**       | 贝叶斯定理、先验与后验、朴素贝叶斯、贝叶斯网络   |

### 第二部分：Neuromatch 实验笔记本

| 周次   | 主题                     | 内容                                               |
| ------ | ------------------------ | -------------------------------------------------- |
| Week 0 | **Python 与数学基础**    | LIF 神经元模型、脉冲与不应期、线性代数与微积分实践 |
| Week 1 | **模型拟合与机器学习**   | 回归、分类、模型选择、交叉验证                     |
| Week 2 | **线性系统与深度学习**   | 线性动力系统、神经网络基础、反向传播               |
| Week 3 | **贝叶斯大脑与强化学习** | 贝叶斯推断、马尔可夫决策过程、强化学习基础         |

### 第三部分：高级主题

| 主题                         | 内容                                             |
| ---------------------------- | ------------------------------------------------ |
| **马尔可夫过程与卡尔曼滤波** | 隐马尔可夫模型 (HMM)、卡尔曼滤波、前向推断与平滑 |

## 前置要求

- **编程**：Python 基础（NumPy、Torch）
- **数学**：高中数学基础
